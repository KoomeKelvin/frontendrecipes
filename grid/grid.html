Grid the masterpiece for responsive web design.

It goes without saying that today, with grid and flex box in CSS today responsive designs are possible without breaking a sweat. The difference between grid and flex box is subtle as both are used in making sites responsive, however grid is majorly for multidimensional that is two dimensional designs whereas flex box is single dimensional it is either items are moving on the main axis or the cross axis.

Autofitand autofill in grid systems.

Autofit and autofill are both used in grids and come in handy when one is setting the grid-template-columns which is a powerful feature that enables setting of columns. autofit and autofill will always be the first parameter in the repeat function and the second parameter will likely be the minmax() which specifies the minimum and maximum width of the colums based on various viewports. When the viewport gives room for extra space that another column maybe added the autofill will fill in the spaces with empty columns whereas the autofit will collapse the spaces making the columns expand to fit in the space that was extra.
